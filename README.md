# Coffee-App

Wisebits FE test task

The application is available at https://ivan-belenkov.github.io/coffee-app/

## Installation

Use the package manager [npm](https://www.npmjs.com/)

```bash
npm install
```

or the package manager [Yarn](https://yarnpkg.com/)

```bash
yarn install
```

to install dependencies.

## Usage

### Production Build

```bash
npm run build:prod
```

or

```bash
yarn build:prod
```

### Development Build

```bash
npm run build:dev
```

or

```bash
yarn build:dev
```

### Start Development Server

```bash
npm start
```

or

```bash
yarn start
```

### Run Tests

```bash
npm test
```

or

```bash
yarn test
```
